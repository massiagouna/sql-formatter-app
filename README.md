# Formateur SQL

Application Streamlit qui corrige automatiquement du code SQL selon les normes
de l'equipe (definies dans `.sqlfluff`), via sqlfluff.

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploiement

Deploye sur [Streamlit Community Cloud](https://share.streamlit.io) en pointant
sur ce repo, fichier principal `app.py`.
